🎲 Pig Dice Game (Python + Tkinter)

A simple yet engaging implementation of the classic Pig dice game, built using Python and Tkinter. This project was created as part of a school computer science assignment and focuses on GUI development, game logic, and user interaction.

🧠 About the Game

Pig is a turn-based dice game where players try to reach a target score (usually 100 points) before their opponent.

🕹️ How It Works
Players take turns rolling a single die.
Each roll adds to the player's current turn score.
A player can choose to:
Roll again and risk losing their turn score
Hold to add the turn score to their total score
⚠️ If a player rolls a 1, their turn ends and they lose all points accumulated during that turn.
The first player to reach the winning score wins!
🚀 Features
🎨 Simple and clean GUI using Tkinter
🎲 Random dice rolling logic
🔄 Turn-based system for two players
🧮 Score tracking (current + total)
🏆 Win condition detection
🖱️ Interactive buttons for rolling and holding
🛠️ Technologies Used
Python
Tkinter (built-in GUI library)
random module (for dice rolls)
